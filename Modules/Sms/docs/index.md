# Sms

短信组件，支持阿里云、腾讯云、七牛<br>
本模块基于 [EasySms](https://github.com/overtrue/easy-sms) 封装。<br>
本模块 `没有` 覆盖容器中的 `EasySms` 类，你需要使用本模块提供的 `Modules\Sms\Services\SmsService` 来发送短信，本服务加载系统配置。
